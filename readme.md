## Installation

1. Clone the repository:
```bash
git clone https://github.com/vitucordeiro/challenge_web_scrapping
cd challenge_web_scrapping
```

2. Install dependencies
```bash
npm install
```
## Usage
Run the scraper:
```bash
npm run start
```
